Web Resources

Let's get coding! (੭•̀ω•́)੭

w3schools Free HTML/CSS tutorials. Where I learned how to build this website and where I continue to go back to for reference.

VisualStudio Code Code editor. Useful plugins: *live server (it emulates a local server on your computer so you can see how your site will look like before updating it to the online server) and *prettier (to see the code better).
I personally use VSCodium, the freely-licensed/Microsoft-telemetry-free distribution of this software.

neocities.org Free site hosting. Used to host this very site! Excellent to experiment. It also has social features and you can explore other personal websites as they are being updated.

Zonelets Simple free blogging engine you can implement in you website. What I first used for my "blog" section.

status.cafe Simple and useful tools to update statuses. Customizable and with a widget you can paste on your website.

bearblog Simple and light blogging platform. It has a discover page where you can read some posts and blogs.

robot.txt How to set up robot.txt to block some or all internet robots from crawling your site. (the ones that listen to the .txt file anyway)

RSS guide How to add a RSS feed to your website. Another, maybe simpler guide can be found at rssguide.neocities.

validator.w3 Nu Html Checker. It checks if your HTML file has some issues to help you better formatting. Useful especially when starting coding so you can catch mistakes early.

WAVE Accessibility WAVE Web Accessibility Evaluation Tools. It checks your pages and tells you how you can make it more accessible. Uselful to use at the beginning of your coding journey as it can point out rookie mistakes.

dead link checker Checks if you are linking to any removed page.

freezeframe.js Script to pause .gifs' animation. I use it on my [links] page to pause the buttons' animations. I followed this tutorial from Bechno Kid to make it work on my site. At first the script was making the images enourmous, but I tweaked the code and now it looks fine. If you have the same problem as me copy my script and CSS. Maybe it'll help!

fontsquirrel Webfont generator. It converts .ttf/.otf/etc font files into .woff & .woff2 to use them on your site.

RGB color picker RGB color codes picker. Similar to w3schools colors picker.

contrastchecker Check if the colour contrast between foreground(text) and background is enough to make your page easily readable.

tinypng Reduce the size of .jpeg .webp and .png files. Essential to reduce your pages' size and make them load faster. base64 does the same.

exifcleaner Privacy friendly open source software that removes metadata from photos. Supports .png .jpg .gif and .tiff.

picdefacer Image Anonymization - privacy friendly free online tool to blur or pixelate photos and images.
